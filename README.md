# 温泉海外博后机会看板

这是一个公开的中文静态看板，用于跟踪 Wen Quan 海外博士后机会、方向匹配、申请入口和材料准备状态。

网站数据存放在本仓库的 `data/opportunities.json`，页面通过 GitHub Pages 发布。

## 公开数据边界

本仓库只公开学校、中心或导师名称、公开链接、研究方向、申请状态和脱敏后的适配摘要。

不公开邮箱、证件、私人申请草稿、附件路径、内部备注或其他敏感申请材料。
